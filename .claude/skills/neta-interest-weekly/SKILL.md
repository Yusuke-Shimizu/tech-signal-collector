# 週次興味関心まとめ作成

`ideas/daily/YYYYMMDD-interest.md` を1週間分集約し、週を通じて繰り返し浮上したテーマを抽出・統合して `ideas/weekly/YYYYWW-weekly.md` に保存する。

## 実行手順

### 0. 前提情報の読み込み

以下を読み込む:
- `.claude/rules/user-profile.md` — ユーザーの興味領域
- 対象週のinterestメモ全ファイル（`ideas/daily/*-interest.md`）
- 前週のweeklyまとめ（存在する場合）— 文脈の接続のため

### 1. 対象期間の特定

ユーザーの引数: `$ARGUMENTS`

**週の定義**: 月曜日〜日曜日を1週間とする（ISO 8601準拠）。

**対象期間の決定手順**:
1. 以下のBashコマンドで対象週の月曜日と日曜日を算出する:
   ```bash
   # 引数なしの場合: 実行日を含む週の月曜日〜日曜日
   date -v-mon -v0d +%Y%m%d  # macOSの場合は以下を使用
   python3 -c "
   from datetime import date, timedelta
   import sys
   today = date.today()
   if len(sys.argv) > 1:
       week_num = int(sys.argv[1])
       monday = date.fromisocalendar(today.year, week_num, 1)
   else:
       monday = today - timedelta(days=today.weekday())
   sunday = monday + timedelta(days=6)
   print(f'MONDAY={monday.strftime(\"%Y%m%d\")}')
   print(f'SUNDAY={sunday.strftime(\"%Y%m%d\")}')
   print(f'WEEK={monday.isocalendar()[1]:02d}')
   print(f'YEAR={monday.isocalendar()[0]}')
   " [週番号]
   ```
2. 算出した月曜日〜日曜日の範囲内にある `ideas/daily/YYYYMMDD-interest.md` を対象とする

**引数の解釈ルール**:
- **週番号指定**（例: `07`）→ その年のISO週番号の月曜日〜日曜日に対応するinterestメモを集約
- **引数なし** → 実行日を含む週（月曜日〜日曜日）を対象。日曜日以降のデータがない場合はそこまでで集約

### 2. interestメモの読み込みと分析

対象期間の全interestメモを読み込み、以下の観点で横断分析:

**テーマ抽出**:
- **複数日にまたがるテーマ**: 2日以上登場したトピックを「週次テーマ」として抽出
- **登場頻度の高いテーマほど上位**に配置
- 単日限りの話題は、インパクトが大きい場合のみ含める

**分析の深さ**:
- 各テーマについて、日ごとの進化・深化を時系列で追跡
- 複数ソース（はてブ、HN、Reddit、AWS等）の横断的な状況を統合
- コミュニティの温度感の変化（投票数・コメント数の推移）
- ユーザープロファイルの興味領域との関連度

### 3. ウィークリーまとめの作成

以下の構成で出力:

**テーマ構成ルール**:
- 5〜8テーマ程度
- 各テーマに「登場日」を明記し、週を通じた持続性を可視化
- 日ごとの進化を表（テーブル）で整理
- 核心的洞察を太字で強調

**最終セクション「週次メタ分析」**:
- 週全体を貫くメタテーマの抽出
- 来週以降の継続ウォッチテーマ
- 自分へのアクション（優先順）

### 4. 出力

**まず「ウィークリーまとめ作成完了。」というメッセージを返してから、結果を保存。**

ファイル: `ideas/weekly/YYYYWW-weekly.md`
- YYYY = 年、WW = ISO週番号（例: 202607）

## 注意事項

- interestメモが存在しない日はスキップ
- 対象期間にinterestメモが1つも存在しない場合は、先に `/neta-interest-daily` の実行を提案する
- 前週のweeklyまとめが存在する場合、テーマの継続性を接続する
- **日次メモの詳細を再掲するのではなく、週を通じた「進化・深化・パターン」を抽出する**ことが目的
- テーマの順序は「登場頻度 × インパクト」で決定
