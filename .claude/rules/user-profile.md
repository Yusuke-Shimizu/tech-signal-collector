# ユーザープロファイル

## 職種・立場
- SIer（システムインテグレーター）のエンジニア
- AI活用を推進しつつ、経営層への説得や組織への浸透も考える立場

## 興味領域

### 最重要: 生成AI/LLMの実践活用
- コーディングエージェント（Claude Code、Codex等）の使いこなし・ワークフロー設計
- エージェント設計パターン、マルチエージェント、並列エージェント
- ローカルLLM（GPU不要での実行、Ollama、RAG構築）
- 各モデルの性能比較と使い分け（Claude vs GPT vs OSS）

### 重要: AI時代のキャリア・組織論
- 「AIで人を減らす」vs「AIで人の価値を上げる」の議論
- SIerとしてのAI活用戦略（コスト削減ではなく利益率改善・高単価シフト）
- 企業のAI教育・トレーニング不在問題と、それを商機にする視点
- 経営層のAI万能論に対する現場目線の反論フレームワーク
- 暗黙知（要件定義、アーキテクチャ判断、障害切り分け）はAIで代替できないという立場

### 重要: 生産性・働き方
- シングルタスク・Deep Workによる集中力の確保
- 「AIが定型作業を処理する分、人間は深く考える時間を確保すべき」という思想
- コンテキストスイッチの排除

### 関心あり: AWS/クラウドインフラ
- Bedrock、SageMaker等のAI/MLサービス
- サーバーレス、コンテナ

### 関心あり: OSS開発/コミュニティ
- OSSの哲学・コミュニティ運営
- Build in Public

### 関心あり: 個人開発/SaaS運営
- Technical SEO、グロースハック、収益化
- 個人開発の成功事例

### 関心あり: JavaScript/TypeScript技術スタック
- 新しいツール、フレームワーク、エディタ
